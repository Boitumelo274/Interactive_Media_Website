# MeloCreates - Music & Hip-Hop Culture Website

## Overview

MeloCreates is a personal portfolio website showcasing the influence of Hip-Hop music on identity,and personal expression. The website features artist spotlights, album recommendations, and a music influence timeline.

## Live Demo

[View Live Site](https://boitumelo274.github.io/Interactive_Media_Website/)

## Pages

### Home Page

- Hero section with tagline
- Introduction about Hip-Hop's influence
- Navigation to all pages

### About Me Page

- Personal music journey
- Childhood inspirations (collapsible section)
- Favorite artists and album influences (collapsible section)
- Profile image

### Music & Hip-Hop Culture Page

- Favorite artists cards (dynamically generated)
- General albums that shaped me
- Music influence timeline with interactive cards
- Modal popups for detailed information
- Tooltips on timeline cards

### Contact Page

- Contact form with validation
- Name, email, and message fields
- Real-time error messages
- Form submission prevention until validation passes

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Custom styling, responsive design, animations
- **JavaScript** - Dynamic content generation, form validation, modal functionality
- **Font Awesome** - Icons for navigation mobile menu
- **Google Fonts** - Inter and Montserrat Fonts

## Features

### Responsive Design

- Mobile-first approach
- Breakpoints for phone, tablet, and desktop
- Hamburger menu for mobile devices
- Grid and flexbox layouts

### Interactive Elements

- Collapsible content sections on About page
- Modal popups for timeline cards
- Form validation with visual feedback
- Hover effects on cards and links
- Mobile menu with overlay and blur effect

### Dynamic Content

- Artist cards generated from JavaScript objects
- Album cards generated dynamically
- Timeline cards with descriptions

## Installation

Clone the repository:

```bash
git clone https://github.com/Boitumelo274/Interactive_Media_Website.git
```

## Alternative

- Clone via GitHub desktop

## Navigate through the project folder

```bash
cd Interactive_Media_Website
```

- Open index.html in your browser

## Deployment

- The site is deployed using GitHub Pages. Any push to the main branch automatically updates the live site.
